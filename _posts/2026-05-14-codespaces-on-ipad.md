Setting up a professional development environment on an iPad used to be difficult. However, by using **GitHub Codespaces**, I can now:

1.  Access a full VS Code instance in Safari.
2.  Run terminal commands for Git and Jekyll.
3.  Edit my portfolio site without needing a laptop.

This setup is ideal for quick updates to my **Today I Learned** section while traveling between Virginia and Colorado.
